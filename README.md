# Say-Bart-Word

A react app created with vite.js


## Run Locally

Clone the project

```bash
  git clone https://github.com/FedericoMp/say-bart-word.git
```

Go to the project directory

```bash
  cd say-bart-word
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```
